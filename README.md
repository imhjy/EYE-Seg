# EYE-Seg


## Introduce

EYE-Seg is a high-resolution image dataset proposed for the field of retinal vascular segmentation, which aims to solve the bottleneck of scarce high-resolution annotated data in this field. The dataset contains 40 collected fundus images, all of which include labels and masks, with a resolution of 2676×2676 pixels each, providing important resources for deep learning model training and vascular morphology research.
